# Influencer Market Map

Интерактивная карта рынка инфлюенс-агентств, построенная с использованием React, D3.js и Tailwind CSS.

## Как запустить

Этот проект не требует установки зависимостей или сборки (build), так как все библиотеки подключаются через CDN, а JSX компилируется прямо в браузере (через Babel Standalone).

1. **Просто откройте файл `index.html`** в любом современном веб-браузере (Chrome, Safari, Firefox, Edge).
   - Можно дважды кликнуть по файлу в папке.
   - Или перетащить файл в окно браузера.

## Структура файлов

- `index.html`: Основной файл для запуска. Содержит HTML-разметку и встроенный код React-компонента. Готов к использованию.
- `influencer-market-map.jsx`: Исходный код React-компонента. Содержит ту же логику, что и `index.html`, но с синтаксисом ES6 modules (`import`/`export`). Предназначен для использования в средах сборки (например, Create React App, Vite), если вы захотите перенести этот код в более крупный проект.

## Как тестировать

Поскольку автоматические тесты отсутствуют, тестирование проводится вручную:

1. **Проверка отображения**:
   - Откройте `index.html`.
   - Убедитесь, что заголовок "Influencer Marketing Market Map" отображается корректно.
   - Проверьте, что график (Scatter Plot) отрисовался и видны точки.

2. **Интерактивность**:
   - **Переключение видов**: Попробуйте переключить режимы "Scatter Plot", "Network", "Grid".
     - В режиме "Network" граф должен перестроиться в сетевую структуру (ego-граф вокруг GRAIL Talent).
     - В режиме "Grid" должен отобразиться список карточек.
   - **Фильтры**: Используйте выпадающие списки (Type, Price, Region, Tech) для фильтрации компаний.
   - **Поиск**: Введите название компании в поле поиска (например, "Viral").
   - **Детали**: Кликните на любую точку или карточку компании, чтобы открыть модальное окно с подробной информацией.
